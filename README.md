# 🚌 Suhana Safar – Smart Bus Transport Management System

Suhana Safar is a **full-stack MERN web application** designed to improve public transportation experience by providing **real-time bus tracking, booking management, schedules, emergency assistance, and user support through an interactive chatbot**.

The platform helps passengers easily track buses, book seats, manage schedules, and access emergency contacts — all from a single dashboard.

---

# 🚀 Features

### 📍 Real-Time Bus Tracking

* Live bus location tracking on an interactive map
* Simulated real-time bus movement
* Route visualization with driver and fare details

### 🎫 Bus Booking System

* Book seats for available buses
* View confirmed bookings
* Cancel bookings when needed

### 📅 Bus Schedule Management

* View available bus routes and schedules
* Organized route and time details

### 🚨 Emergency Assistance

* Quick access to important emergency numbers
* Dedicated emergency support section

### 📞 Personal Contact Management

* Save and manage important contacts
* Easy access during emergencies

### ⭐ Feedback System

* Submit feedback about transport services
* Helps improve the platform

### 🤖 Smart Chat Assistant

An in-app assistant that helps users with:

* Booking guidance
* App navigation
* Route and schedule information
* Emergency contact support

### 📊 User Dashboard

Provides an overview of:

* Total buses
* Active buses
* Confirmed bookings
* User activity

### 🔐 Authentication System

* Secure user signup and login
* JWT-based authentication

### 🌗 Light / Dark Mode

* User-friendly theme toggle for better accessibility

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```
git clone https://github.com/your-username/suhana-safar.git
cd suhana-safar
```

---

## 2️⃣ Backend Setup

```
cd server
npm install
```

Create a `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run backend:

```
npm run dev
```

---

## 3️⃣ Frontend Setup

```
cd client
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```
